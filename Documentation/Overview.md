# AvansMatrixBoard – Project Overview

## What is AvansMatrixBoard?

AvansMatrixBoard is a digital LED matrix display system used within Avans school. It is capable of displaying traffic-style messages as well as images and custom visual content. The system is controlled through a web-based interface and runs on a Raspberry Pi connected to the LED matrix panel.

## System Components

The project consists of four main parts:

1. Frontend (Web Interface)
2. Backend (Flask API)
3. Raspberry Pi Controller
4. LED Matrix Board

## Main Goals

- Provide an easy way to control the LED matrix
- Display real-time information and visuals
- Allow future students to expand functionality
